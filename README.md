# Perfume Store

Premium Angular standalone storefront for a luxury men's fragrance shop.

## Что сделано

- Каталог из 15 мировых брендов мужского парфюма.
- Тёмная premium тема, минималистичный интерфейс.
- Фильтрация по брендам.
- Админ-панель для добавления, редактирования и удаления товаров.
- Корзина с изменением количества, удалением и подсчётом суммы.
- Оформление заказа с формой имени, телефона и адреса.
- Псевдо-отправка заказа в mock Telegram сервис.
- Сохранение состояния каталога, корзины и заказов в `localStorage`.

## Структура

- `frontend/` — Angular приложение.
- `frontend/src/app/components/` — UI-компоненты.
- `frontend/src/app/services/` — сервисы для продуктов, корзины, заказов и Telegram.
- `DESIGN.md` — дизайн-система, на которой основан UI.
- `SPEC.md` — техническая спецификация проекта.

## Запуск

```bash
cd frontend
npm install
npm run start
```

Откройте `http://localhost:4200`.

## Сборка

```bash
cd frontend
npm run build
```

## Примечания

- Удалённые товары не возвращаются после перезагрузки.
- Заказы сохраняются и имитируются через mock Telegram сервис.
